### Full Documentation:

Proper documentation is built using mkdocs and located under `./Docs/`. Read more there

**To get Started with docs**

install mkdocs 
```bash
pip install mkdocs
```

navigate to `./Docs`
```bash
cd Docs
```

start the docsite
```bash
mkdocs serve
```
